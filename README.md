Program pentru realizarea agendelor (C# .NET)

Prin selectarea unui interval (o dată de început și una de final), urmată de apăsarea butonului „Start!” și alegerea unui folder, se va salva automat câte o pagină (poză png) pentru fiecare zi din intervalul selectat. (vezi exemplu în folderul „Exemplu agenda”)

Pot fi alese culorile pentru text și fundal și se poate pune o imagine pentru fundalul paginilor.

Deasemenea, se pot introduce sărbători anuale și lunare. După apăsarea butonului „Ok”, pentru acele zile va apărea pe paginile corespunzătoare și mesajul „La mulți ani!”.

Răsfoirea agendei se face prin butoanele cu săgeți de pe fereastră, sau prin cele din tastatură.
